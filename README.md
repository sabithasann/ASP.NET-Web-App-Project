# ASP.NET-Web-App-Project
 Developed a robust doctor’s private chamber management platform with user-friendly interfaces for appointment
scheduling, patient records, creating prescriptions, administrative tasks, etc. There are three types of users:
Patient, Doctor, and Assistant.
• Use the MVC framework for the whole project. In addition, dynamic search and filtering capabilities were
implemented using AJAX and jQuery.
• Designed Web pages with HTML, CSS, and JavaScript, ensuring a smooth user experience for patients and
clinic staff.
